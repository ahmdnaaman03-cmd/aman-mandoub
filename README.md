ok# 🛡️ نظام أمان المندوب (Aman Mandoub System)

       https://ahmdnoaman.pythonanywhere.com/
نظام بسيط وفعال لإدارة عمليات تحصيل المبالغ المالية للمندوبين عبر رموز الاستجابة السريعة (QR Code). يتيح النظام للمندوب الاستعلام عن الشحنة وتوليد كود دفع خاص للعميل، مع إمكانية متابعة حالة الدفع في الوقت الفعلي.

## 🚀 المميزات
- **الاستعلام عن الشحنات:** بحث سريع برقم الشحنة.
- **توليد QR Code:** إنشاء رمز دفع فريد لكل عميل.
- **بوابة دفع العميل:** واجهة بسيطة للعميل لتأكيد الدفع.
- **تغيير الحالة يدوياً:** إمكانية تبديل حالة الدفع من قبل المندوب.
- **التقرير اليومي:** عرض جدول كامل لجميع الطلبات وحالات التحصيل.
- **تصميم متجاوب:** واجهات حديثة تعمل على الموبايل والكمبيوتر.

## 🛠️ التقنيات المستخدمة
- **Python (Flask):** لإدارة العمليات والمسارات.
- **HTML/CSS:** لتصميم الواجهات.
- **QRcode Library:** لتوليد رموز الاستجابة السريعة.
- **CSV:** لتخزين بيانات الشحنات بشكل بسيط.

## 📂 هيكل المشروع
- `app.py`: الملف الأساسي الذي يحتوي على منطق التطبيق والمسارات.
- `templates.py`: يحتوي على قوالب HTML للواجهات (المندوب، العميل، التقرير).
- `data.csv`: قاعدة بيانات بسيطة تحتوي على أرقام الشحنات، أسماء العملاء، والأسعار.
- `requirements.txt`: قائمة المكتبات اللازمة لتشغيل المشروع.

## ⚙️ كيفية التشغيل
1. قم بتحميل المستودع.
2. ثبت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
3. شغل التطبيق:
   ```bash
   python app.py
   ```
4. افتح المتصفح على الرابط: `http://localhost:8000`

## 📝 ملاحظة
يستخدم المشروع متغير `GLOBAL_URL` في ملف `app.py`. تأكد من تحديثه برابط الاستضافة الخاص بك (مثل Koyeb أو Render) لضمان عمل الـ QR Code بشكل صحيح عند مسحه من قبل العميل.
